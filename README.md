# BudgetFlow — Budget Tracker App

A personal finance web app that helps you track expenses, manage budgets, and stay on top of your money. Built with Next.js and protected behind authentication so your data stays yours.

## Live Demo

[budget-tracker-pty7.vercel.app](https://budget-tracker-pty7.vercel.app)

## Features

- User authentication — only logged-in users can access data
- Dashboard with total balance, total spent, and remaining budget
- Recent budgets and transactions shown on dashboard
- Spending overview graph
- Create and manage multiple budgets
- Add and view transactions
- Clean, responsive UI

## Tech Stack

- Next.js
- JavaScript
- Tailwind CSS

## Getting Started

```bash
git clone https://github.com/Mehdi-Chandia/BudgetTracker.git
cd BudgetTracker
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
